# Bank Analysis Dashboard – Power BI | Excel | SQL | Tableau

## Project Overview
The **Bank Analysis Project** provides a complete end-to-end analysis of customer loans, interest rates, loan purposes, verification status, home ownership, and state-wise loan performance.  
This multi-tool project includes **Excel data cleaning**, **SQL data analysis**, and **interactive dashboards in Power BI and Tableau**.

It helps financial institutions understand:
- Loan trends  
- Customer segmentation  
- Risk levels  
- Repayment behavior  

This project is ideal for **Data Analytics, Business Intelligence, and Financial Analytics** roles.

## Project Components

### Excel
- Raw dataset cleaning  
- Missing value handling  
- Removing duplicates  
- Preparing tables for SQL & BI tools  

### SQL
- Customer and loan data analysis  
- Joins, aggregations, window functions  
- KPIs generated using SQL queries  
- Segmentation of customers by:
  - Loan status  
  - Credit grade  
  - Loan purpose  

### Power BI
- Fully interactive dashboard  
- DAX-based KPIs  
- Trend analysis & drill-down interactions  

### Tableau
- Additional visual storytelling  
- State-wise and grade-wise analysis  
- Interactive filtering & map visuals
